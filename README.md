 Features

CodeQuest is built to provide a structured, gamified learning experience:

Gamified Learning Paths: Progress through structured levels and quests that introduce DSA concepts incrementally.

Interactive Challenges: Solve problems directly in an embedded code editor with support for multiple languages.

Real-time Feedback: Instantly verify your solutions against comprehensive test cases.

DSA Topic Coverage: Covers essential topics including Arrays, Linked Lists, Trees, Graphs, Sorting, and Dynamic Programming.

Progress Tracking: Monitor your mastery, view achievements, and see where you rank among other learners.

Responsive Design: Learn anytime, anywhere, on desktop or mobile devices.
